# Bayesian Data Analysis Project
Project

# Getting started
```
python -m venv venv
source venv
pip install -r requirements.txt
python test.py
```

# Fetching data
In project root, run `bin/download_data.sh`.
This requires [jq](https://stedolan.github.io/jq/).
